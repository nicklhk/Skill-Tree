# C++基础知识

## 1. C++的公有继承、保护继承和私有继承

|      |  public   | protected | private |
| :--: | :-------: | :-------: | :-----: |
| 公有继承 |  public   | protected |   不可见   |
| 保护继承 | protected | protected |   不可见   |
| 私有继承 |  private  |  private  |   不可见   |
